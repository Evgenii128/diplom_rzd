## ✨ Функциональные возможности

Система состоит из двух основных частей: публичной (для клиентов) и административной (для сотрудников).

### 👥 Публичная часть (Клиенты)

| Функция                   | Описание                                                                                      |
|---------------------------|-----------------------------------------------------------------------------------------------|
| **Главная страница**      | Презентация компании с кратким описанием услуг и преимуществ                                  |
| **Подача заявления**      | Интерактивная форма с выбором услуг, указанием количества и автоматическим расчетом стоимости |
| **Отслеживание статуса**  | После отправки заявки пользователь получает уникальный номер для отслеживания                 |

### 🔐 Административная панель (Сотрудники)

#### 📊 Дашборд
- Ключевые показатели: выручка, активные договоры, счета к оплате, количество клиентов
- Статистика по заявкам с визуальным распределением по статусам
- Список последних заявок и ближайших дедлайнов по договорам
- График активности за 7 дней
- Быстрые действия: создание услуги, клиента, договора или счета

#### 📝 Управление заявками
- Просмотр всех входящих заявок с детальной информацией
- Фильтрация по статусу (новые, в работе, завершенные, отклоненные)
- Изменение статуса и добавление комментариев
- Создание договора напрямую из заявки в один клик

#### 🛠️ Управление услугами
- Полный CRUD (создание, просмотр, редактирование, удаление) услуг
- Указание кода услуги, названия, описания, единицы измерения и цены
- Поиск и фильтрация по активности
- Автоматическая проверка уникальности кода услуги

#### 👤 Управление клиентами
- Ведение базы клиентов с полной контактной информацией
- Хранение юридических адресов и банковских реквизитов
- Поиск по названию, контактному лицу, email или телефону

#### 📄 Управление прайс-листами
- Создание прайс-листов с указанием периода действия
- Добавление позиций с выбором услуги и индивидуальной ценой
- Автоматическое заполнение базовой цены из справочника услуг
- Просмотр детальной информации по каждому прайс-листу

#### 📑 Управление договорами
- Создание и ведение договоров с привязкой к клиенту
- Отслеживание статуса (активен, приостановлен, завершен)
- Контроль сроков действия с визуальным выделением истекающих
- Фильтрация по статусу
- Быстрое выставление счета из договора

#### 💳 Управление счетами
- Выставление счетов на основе договоров
- Автоматический расчет общей суммы
- Отслеживание просроченных платежей (визуальное выделение)
- Отметка об оплате с фиксацией даты
- Фильтрация по статусу (ожидают, оплачены, отменены, просрочены)

#### 📋 Аудит
- Автоматическое логирование всех ключевых действий пользователей
- Фиксация времени, типа операции и затронутых таблиц

## 🛠️ Технологический стек

### Бэкенд
- **Runtime**: [Node.js](https://nodejs.org/) (версия 16+)
- **Фреймворк**: [Express.js](https://expressjs.com/ru/) (версия 4.22)
- **База данных**: [MySQL](https://www.mysql.com/) с использованием `mysql2/promise` (версия 3.18)
- **Аутентификация**: 
  - `bcrypt` (версия 6.0) - хеширование паролей
  - `jsonwebtoken` (версия 9.0) - JWT токены
  - `cookie-parser` (версия 1.4) - управление куками

### Фронтенд
- **Разметка**: HTML5
- **Стилизация**: CSS3 с поддержкой светлой/темной темы
- **Логика**: JavaScript (ES6+)
- **Иконки**: [Font Awesome 6.4.0](https://fontawesome.com/)
- **Шрифты**: [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter)

### Инструменты разработки
- **Nodemon** (версия 3.0) - автоматический перезапуск сервера
- **CORS** - обеспечение кросс-доменных запросов
